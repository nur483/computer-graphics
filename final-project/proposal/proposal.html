Motivation - Perfectly Timed

The above image shows a picture of an avalanche taken in the exact right moment. A few instances earlier or later, we would have only seen a snow covered slope. Hence, it directly conforms with the required theme aka. "Perfectly Timed".

Feature Breakdown

The most important feature for this scene is the participating media, which is needed to model the avalanche. Also important is a good BSDF that captures snow well. To build a realistic scene, an environment map emitter is used for the mountain panorama.

0 pts Modeling Meshes

One side of a mountain modeled in Blender. This will be used as the slope in the foreground where the avalanche goes down. I assigned zero points to this task, to stay within the 60 points limit. I did not drop this task, since it is needed for the motivational image.

Validation

Not needed

15 pts Environment Map Emitter

A standard environment map emitter, nothing more nothing less.

Validation

Rendering a standard object (e.g. teapot) and comparing the result to outputs produced by the open source renderer Mitsuba.

15 pts Disney BSDF

Implementation of 5 parameters of the Disney BSDF. I have not decided which parameters to implement, but probably the ones most relevant for snow.

Validation

Rendering spheres with different paramters and comparing them to the some reference image (e.g. results from paper).

30 pts Heterogeneous Volumetric Participating Media

The main part of this project, using volumetric path tracing (either delta tracking or ray marching). I will implement an isotropic phase function and no emission. For the participating media I will use volume discretization (e.g. a voxel grid).

15 pts Subgoal Homogeneous Volumetric Participating Media

As a first step, I will implement a homogeneous version. Reaching this subgoal will only account for 15 out of 30 points.

Validation

Rendering a cornell box with a (heterogeneous) volumetric participating media. Comparing results to renders of the same scene with Mitsuba.
